## Install

```bash
sudo curl \
  -L "https://github.com/takaaki-kasai/git-foresta/raw/master/git-foresta" \
  -o "/bin/git-foresta"
  && sudo chmod +x "/bin/git-foresta"
```

## Usage

```bash
git-foresta --all --style=15  | less -RSX
```
